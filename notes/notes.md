

##### 默认配置文件路径：

- 配置文件：/etc/my.cnf
- 日志文件：/var/log/mysqld.log
- 服务启动脚本：/usr/lib/systemd/system/mysqld.service
- socket文件：/var/run/mysqld/mysqld.pid

- source  ~/.bash_profile    -重启是配置生效

##### yum

1. 使用YUM查找软件包 
   ：yum search 

2. 列出所有可安装的软件包 
   ：yum list 

3. 列出所有可更新的软件包 

    ：yum list updates

4. 列出所有已安装的软件包 
   ：yum list installed 

5. 列出所有已安装但不在 Yum Repository 内的软件包 
   ：yum list extras 

6. 使用YUM获取软件包信息 
   ：yum info 

7. 列出所有可更新的软件包信息 
   ：yum info updates 

8. 列出所有已安装的软件包信息 
   ：yum info installed 

9. 列出所有已安装但不在 Yum Repository 内的软件包信息 
   ：yum info extras 

10. 列出软件包提供哪些文件 
    ：yum provides
